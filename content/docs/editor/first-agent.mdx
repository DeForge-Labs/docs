## Prerequisites
Before you begin, make sure you have a Deforge account and a Telegram bot token. If you don't have a Deforge account yet, you can [sign up for free](https://app.deforge.io/). To create a Telegram bot, message `@BotFather` on Telegram and follow the instructions to get your bot token.
