## Description
Google Veo node allows you to create AI videos using Google Veo2 or Veo3, the choice upto you. The node also has plethora of options to configure your AI video generation. Some options are exclusive to Veo2 and some to Veo3. They are: - Duration: only applicable to Veo2 (can choose from 5 to 8 seconds. Veo3 can only generate 8 seconds videos) - Ratio: only applicable to Veo2 (can also create 9:16 videos while Veo3 can only generate 16:9 videos) - Generate Audio: only applicable to Veo3 (only Veo3 can generate audio along with video).

### Common Use Cases
- Can be used to create automated video uploads to YouTube.
- Create videos via telegram messages and easily share among people.
- Create videos for work purposes, privately.
