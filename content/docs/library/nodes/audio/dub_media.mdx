## Description
The Dub Video Audio node allows you to dub the audio of a video or audio file into another language using the Eleven Labs API.

### Common Use Cases
- Dub youtube videos to your language for easier consumption
- Dub ad campaigns to multiple languages
- Use as a realtime translator agent
