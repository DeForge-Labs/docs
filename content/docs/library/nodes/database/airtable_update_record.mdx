## Description
This node allows you to update a record in Airtable. You can specify the base, table, and record you want to update, as well as the fields and their new values. This is useful for keeping your Airtable data up-to-date based on actions or events in your workflow.

### Common Use Cases
- Updating a contact's information in Airtable after receiving new data from a form submission.
- Modifying a task's status in Airtable when it's marked as complete in another tool.
- Changing a product's inventory count in Airtable after a purchase is made.
