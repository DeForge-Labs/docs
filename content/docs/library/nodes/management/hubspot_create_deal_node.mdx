## Description
The **HubSpot - Create Deal** node allows you to create a new deal in your HubSpot CRM account. This node is useful for automating sales processes, tracking potential revenue, and managing customer relationships effectively.

### Common Use Cases
- Automating the creation of sales deals based on lead generation activities.
- Integrating with other sales and marketing tools to streamline workflows.
- Tracking deal progress and updating sales pipelines automatically.
