## Description
The Notion List Databases node allows you to fetch a list of all databases in your Notion workspace that are shared with this integration. This is useful for discovering available databases, retrieving their IDs for use in other nodes, and managing your Notion data within your workflows. You can also provide an optional query to filter the databases by name, making it easier to find specific databases in larger workspaces.

### Common Use Cases
- Retrieve a list of Notion databases to display to users or use in subsequent nodes.
- Filter databases by name to find the right one for your workflow.
- Use the database IDs in other Notion nodes to create pages, query data, or update content.
