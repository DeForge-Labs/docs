## Description
The Create Google Sheet node allows you to create a new Google Sheets spreadsheet and populate it with data from a CSV file or raw CSV string. This node is useful for automating the creation of spreadsheets for data analysis, reporting, or sharing information.

### Common Use Cases
- Automatically generate spreadsheets from data collected in other parts of your workflow.
- Create reports in Google Sheets format for easy sharing and collaboration.
- Convert raw CSV data into a structured Google Sheets document.
