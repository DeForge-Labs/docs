## Description
The **Send Telegram Media Message** node allows you to send a media message (such as an image, video, or audio file) to a specified Telegram chat using your Telegram bot. This node is useful for sharing rich media content within your Telegram chats.

### Common Use Cases
- Send media files to telegram
- Send text to speech voice messages via agent
- Send dubbed videos via eleven labs agent
