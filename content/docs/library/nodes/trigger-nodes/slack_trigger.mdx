## Description
The Slack Trigger node allows you to trigger a flow when a message is received by your Slack bot. This is useful for responding to user messages, processing commands, or integrating with other services based on Slack interactions.

### Common Use Cases
- Responding to user messages in Slack channels or direct messages.
- Processing commands sent to the bot.
- Integrating Slack interactions with other services or workflows.
