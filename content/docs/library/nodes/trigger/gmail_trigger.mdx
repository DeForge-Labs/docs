## Description
The Gmail Trigger node allows you to trigger a flow when a new email is received in your Gmail account. This node listens for incoming emails and initiates the flow execution whenever a new email arrives, enabling you to automate actions based on email events.

### Common Use Cases
- Automatically send a reply or acknowledgment when a new email is received.
- Extract specific information from incoming emails and use it in subsequent nodes.
- Create tasks or tickets in project management tools based on email content.
